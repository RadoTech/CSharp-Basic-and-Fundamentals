4.	Array Rotation
Write a program that receives an array and number of rotations you have to perform (first element goes at the end) Print the resulting array.
Examples
Inputs(3)	                

51 47 32 61 21

2
--------------------
32 21 61 1

4
--------------------
2 4 15 31

5

Outputs(3)
32 61 21 51 47
32 21 61 1
4 15 31 2
